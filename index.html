Alerts

Information alert

This is the information alert

× Alert! Information alert

HTML

            
                <div class="alert info">
                    <span class="close-btn" onclick="this.parentElement.
                    style.display='none';">& times;</span> 
                    <strong>Alert!</strong> Information alert
                </div>

CSS

            
                .alert {
                padding: 20px;
                color: white;
                font-family: 'Helvetica', sans-serif;
                font-size: 18px;
                line-spacing: 1px;
                }

                .info {
                background-color: #91C8D6;
                }
                
                .close-btn {
                margin-left: 15px;
                color: white;
                font-weight: bold;
                float: right;
                font-size: 22px;
                line-height: 20px;
                cursor: pointer;
                transition: 0.2s;
                }

                .close-btn:hover {
                color: #E0E0E0;
                }
