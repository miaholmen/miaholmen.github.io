Pattern Library

Buttons

Buttons can be used to link to a new action, a new page on the website or an external website. They can also be used in forms for submitting data.

Default button

This is the default button which can be used for most actions, although the selection of buttons below is often more appropriate for certain actions.

The buttons all have a font size of 16px and a 1px letter spacing.

HTML

            
                <a href="#">
                <button class="default-btn">Default</button>
                </a>

CSS

            
                .default-btn {
                background-color: #404040;
                border: none;
                color: white;
                width: 160px;
                padding: 12px 0px;
                text-align: center;
                text-decoration: none;
                display: inline-block;
                font-size: 16px;
                letter-spacing: 1px;
                cursor: pointer;
                }

                .default-btn:hover {
                background-color: #5D5C5C;
                }

Alerts

Alert messages are used to notify the user of something; either information, danger/caution or success.

Information alert

This alert will appear when the user needs to be informed about something. The user can click the X to make the message disappear.

× Alert! Information alert

HTML

            
                <div class="alert info">
                    <span class="close-btn" onclick="this.parentElement.
                    style.display='none';">& times;</span> 
                    <strong>Alert!</strong> Information alert
                </div>

CSS

            
                .alert {
                padding: 20px;
                color: white;
                font-family: 'Helvetica', sans-serif;
                font-size: 18px;
                line-spacing: 1px;
                }

                .info {
                background-color: #91C8D6;
                }
                
                .close-btn {
                margin-left: 15px;
                color: white;
                font-weight: bold;
                float: right;
                font-size: 22px;
                line-height: 20px;
                cursor: pointer;
                transition: 0.2s;
                }

                .close-btn:hover {
                color: #E0E0E0;
                }

Call to action

The call to action can be used should there be a need to call extra attention to specific content on the website. It has a blue background to make it stand out and allows for a heading and text. The colour can also be changed depending on the usage.

Call to action

Text for the call to action goes here!

HTML

            
                <div class="cta">
                   <h1>Call to action</h1>
                   <p>Text for the call to action goes here!</p>
                </div>

CSS

            
                .cta {
                font-family: Helvetica, sans-serif;
                background-color: #91C8D6;
                color: #fff;
                width: 420px;
                padding: 40px;
                border-radius: 3px;
                }

                .cta h1 {
                font-size: 42px;
                margin: auto;  
                }
